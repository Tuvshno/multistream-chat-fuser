How to Get Popout Chat URL

For Twitch:

  1. Go to the Twitch stream you want to view.
  2. Below the video, click on the chat settings icon (gear icon).
  3. In the settings menu, click"Popout Chat".
  4. A new window with the chat will open. Copy the URL from the browser's address bar.
  5. Paste the copied URL into the URL Input in the Multistream Chat Fuser.

For YouTube:

  1. Go to the YouTube stream you want to view.
  2. Next to the chat, click on the three dots for more options.
  3. In the menu that appears, click "Popout chat".
  4. A new window with the chat will open. Copy the URL from the browser's address bar.
  5. Paste the copied URL into the URL Input in the Multistream Chat Fuser.
